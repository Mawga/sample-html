Tag Practice

Practice use of various tags including table.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (size dependent on levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header container for introductory content
div division container for block-elements
ol ordered list define an ordered list
ul unordered list define an unordered list
li list item define a list item

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>example</em> example
strong strong make strong text <strong>example</strong> example
a anchor make hyperlink <a href="http://google.com/ target="_blank">example</em> example
img image include an image <img src="images/tv.png" alt="tv" >
span span container for inline-elements <span style="color:red">example</span> example
