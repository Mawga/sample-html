Simple HTML Website

Purpose

To quickly refresh memory on HTML before practicing the creation of more completed websites using CSS and Javascript.
The website uses inline style throughout. The Tags page displays two tables with various block/inline-element tags. The Cow page is a silly page constructed to show use of div and span with inline style.
